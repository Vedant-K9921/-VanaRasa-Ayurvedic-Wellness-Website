# ⚙️ Installation

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/vanarasa-ayurveda.git
```

## 2. Open Project Folder

```bash
cd vanarasa-ayurveda
```

## 3. Run the Website

Simply open:

```bash
index.html
```

Or use VS Code Live Server.

---

# 🎵 Adding Ambient Music (Optional)

Place your audio file inside the root folder:

```bash
ambient.mp3
```

The built-in music toggle button will automatically control playback.

Recommended sounds:
- Forest ambience
- Temple bells
- Meditation flute
- Nature healing sounds
